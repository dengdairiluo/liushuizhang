### 1.1 抽象过程

一种面向对象程序设计方式：
1. 万物皆对象
2. 程序是对象的集合
3. 每个对象都有自己的由其他对象所构成的存储
4. 每个对象都拥有其类型。
5. 某一特定类型的所有对象都可以接收同样的消息
